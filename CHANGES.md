```
قبل: صف أفقي واحد يسبب اكتظاظ على الموبايل
[صورة] [اسم+سعر] [أزرار كمية] [حذف] ← كلهم في صف واحد

بعد: تخطيط متجاوب flex-col sm:flex-row
على xs:                     على sm+:
┌─────────────────┐        ┌──────────────────────────┐
│ [صورة] اسم+سعر   │        │ [صورة] اسم+سعر  [-1+] 🗑️ │
│ [- 1 +]     🗑️  │        └──────────────────────────┘

```

### ٢. نقاط التوقف المتجاوبة المضافة
- العنوان: `text-2xl sm:text-3xl`
- الهوامش: `py-6 md:py-10`, `px-3 sm:px-4`
- البطاقات: `p-3 sm:p-4`
- الأزرار: `h-3.5 w-3.5 sm:h-3 sm:w-3`
- الأيقونات: `h-14 w-14 sm:h-16 sm:w-16`

```tsx
<img src={it.image} alt={it.name} width={64} height={64} loading="lazy" />
```
هذا يمنع Cumulative Layout Shift (CLS).

## 📊 تعليمات ربط Google Sheets

### الخطوة ١: إنشاء Google Apps Script
1. افتح [sheets.new](https://sheets.new) لإنشاء Google Sheet جديد
2. اذهب إلى **Extensions > Apps Script**
3. الصق محتوى ملف `google-apps-script.gs` كاملاً
4. احفظ (Ctrl+S) وسمِّ المشروع "Elysr Webhook"

### الخطوة ٢: النشر كـ Web App
1. في محرر Apps Script، اضغط **Deploy > New Deployment**
2. اختر النوع: **Web App**
3. إعدادات:
   - Execute as: **Me** (حسابك)
   - Who has access: **Anyone** (أي شخص)
4. اضغط **Deploy**
5. انسخ الرابط (يشبه: `https://script.google.com/macros/s/XXXXX/exec`)

### الخطوة ٣: ربط المشروع
1. افتح ملف `src/lib/governorates.ts`
2. استبدل `YOUR_DEPLOYMENT_ID` في السطر:
   ```ts
   const GOOGLE_SHEETS_WEBHOOK_URL = "https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec";
   ```
   بمعرف النشر الحقيقي

### الخطوة ٤: اختبار
1. ارفع التغييرات للموقع (git push)
2. أضف منتج للسلة وأكمل الطلب
3. تحقق من Google Sheet — يجب أن يظهر الطلب في شيت "الطلبات"
4. جرب صفحة الجملة — يظهر في شيت "الوكلاء والتجار"
